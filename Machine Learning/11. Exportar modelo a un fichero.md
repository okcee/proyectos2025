# Exportar modelo a un fichero
Una vez que hemos entrenado nuestro modelo, podemos guardarlo en un fichero binario (mediante la librer√≠a pickle) para poder llevarlo a otro entorno o programa.

**Pasos:**

1.- Importamos la librer√≠a pickle
```python
import pickle
```
2.- Creamos un fichero binario (modelo_entrenado.pickle) con el modelo entrenado
```python
with open("modelo_entrenado.pickle", "wb") as file:
    pickle.dump(modelo_entrenado, file)
```
# Importar modelo desde un fichero
Si queremos importar un modelo de machine learning ya entrenado que est√° grabado en un fichero binario, seguiremos estos pasos:

1.- Importamos la librer√≠a pickle
```python
import pickle
```
2.- Importamos el modelo desde el fichero binario "modelo_entrenado.pickle"
```python
with open('modelo_entrenado.pickle', "rb") as file:
    modelo_entrenado = pickle.load(file)
```

# Laboratorio para guardar y recuperar un modelo de un fichero binario pickle

En el desarrollo de modelos de Machine Learning, una vez que hemos entrenado un modelo y estamos satisfechos con su rendimiento, es fundamental poder guardarlo para su uso futuro. Esto nos evita tener que reentrenar el modelo cada vez que necesitemos hacer predicciones. La librer√≠a `pickle` de Python es una herramienta excelente para serializar (guardar) y deserializar (cargar) objetos Python, incluyendo modelos de Machine Learning complejos.

## Supuesto üìù

Imaginemos que hemos desarrollado un **modelo de regresi√≥n lineal simple** para predecir el precio de una vivienda üè° bas√°ndose en su superficie en metros cuadrados. Hemos entrenado este modelo con un conjunto de datos y ahora queremos guardarlo para poder usarlo m√°s adelante para predecir el precio de nuevas viviendas sin necesidad de volver a entrenarlo.

**Modelo:** Regresi√≥n Lineal
**Variable Independiente (Caracter√≠stica):** Superficie (m¬≤)
**Variable Dependiente (Objetivo):** Precio (‚Ç¨)

---

## 1. Guardar modelo en un fichero üíæ

Una vez que nuestro modelo est√° entrenado, podemos usar `pickle.dump()` para guardarlo en un archivo binario. Es com√∫n usar la extensi√≥n `.pkl` o `.pickle` para estos archivos.

**Pasos:**
1.  Importar las librer√≠as necesarias (`pickle` y la librer√≠a del modelo, por ejemplo, `sklearn.linear_model`).
2.  Entrenar el modelo (en nuestro supuesto, ya lo tendr√≠amos entrenado).
3.  Abrir un archivo en modo escritura binaria (`'wb'`).
4.  Utilizar `pickle.dump(nombre_del_modelo, archivo_abierto)` para guardar el modelo.

**Ejemplo de c√≥digo:**

```python
import pickle
from sklearn.linear_model import LinearRegression
import numpy as np

# --- Simulaci√≥n del entrenamiento del modelo (Supuesto) ---
# Datos de ejemplo: Superficie (m^2) y Precio (‚Ç¨)
X_entrenamiento = np.array([[50], [70], [80], [100], [120], [150]]) # Superficie
y_entrenamiento = np.array([150000, 200000, 220000, 280000, 330000, 400000]) # Precio

# Crear y entrenar el modelo de regresi√≥n lineal
modelo_precios_casas = LinearRegression()
modelo_precios_casas.fit(X_entrenamiento, y_entrenamiento)

print(f"Modelo entrenado. Coeficiente: {modelo_precios_casas.coef_[0]:.2f}, Intercepto: {modelo_precios_casas.intercept_:.2f}")
# --- Fin de la simulaci√≥n del entrenamiento ---

# Nombre del archivo donde se guardar√° el modelo
nombre_fichero_modelo = 'modelo_precios_vivienda.pkl'

# Guardar el modelo
with open(nombre_fichero_modelo, 'wb') as archivo:
    pickle.dump(modelo_precios_casas, archivo)

print(f"Modelo guardado correctamente en el archivo: {nombre_fichero_modelo}")
```

# 2. Crear un nuevo modelo con los datos del fichero üìÇ

Para cargar el modelo previamente guardado, utilizamos `pickle.load()`. Esto deserializar√° el objeto desde el archivo, reconstruyendo el modelo tal como estaba cuando fue guardado.

Pasos:
1. Importar la librer√≠a pickle.
2. Abrir el archivo donde est√° guardado el modelo en modo lectura binaria (`'rb'`).
3. Utilizar `pickle.load(archivo_abierto` para cargar el modelo en una nueva variable.

Ejemplo de c√≥digo:
```Python
import pickle
# No es necesario importar sklearn aqu√≠ si solo cargamos y usamos,
# pero es buena pr√°ctica saber qu√© tipo de objeto se espera.
# from sklearn.linear_model import LinearRegression

# Nombre del archivo donde se guard√≥ el modelo
nombre_fichero_modelo = 'modelo_precios_vivienda.pkl' # Aseg√∫rate que este archivo exista

# Cargar el modelo desde el archivo
with open(nombre_fichero_modelo, 'rb') as archivo:
    modelo_cargado = pickle.load(archivo)

print(f"Modelo cargado correctamente desde el archivo: {nombre_fichero_modelo}")

# Podemos verificar que el modelo cargado tiene los mismos par√°metros (opcional)
# Nota: Necesitar√≠as tener acceso a la clase original del modelo para hacer esto
# o para que el objeto funcione correctamente (pickle lo maneja impl√≠citamente si est√° disponible)
if hasattr(modelo_cargado, 'coef_'):
    print(f"Coeficiente del modelo cargado: {modelo_cargado.coef_[0]:.2f}")
if hasattr(modelo_cargado, 'intercept_'):
    print(f"Intercepto del modelo cargado: {modelo_cargado.intercept_:.2f}")
```

# 3. Utilizar el nuevo modelo üöÄ

Una vez cargado, el `modelo_cargado` es una instancia del modelo original y puede ser utilizado directamente para hacer predicciones o cualquier otra operaci√≥n que el modelo soporte.

Pasos:
1. Preparar los nuevos datos sobre los cuales queremos hacer predicciones. Deben tener el mismo formato que los datos utilizados para entrenar el modelo original.
2. Usar el m√©todo de predicci√≥n del modelo cargado (generalmente .predict()).

Ejemplo de c√≥digo:
```Python
import pickle
import numpy as np
# from sklearn.linear_model import LinearRegression # Necesario si quieres ver el tipo o atributos espec√≠ficos

# Nombre del archivo donde se guard√≥ el modelo
nombre_fichero_modelo = 'modelo_precios_vivienda.pkl'

# Cargar el modelo (asumiendo que ya se ejecut√≥ el paso 2, si no, incluir el c√≥digo de carga)
with open(nombre_fichero_modelo, 'rb') as archivo:
    modelo_cargado = pickle.load(archivo)
print("Modelo listo para usar.")

# Nuevos datos para predicci√≥n: una casa de 90 m^2 y otra de 130 m^2
nuevas_superficies = np.array([[90], [130]])

# Realizar predicciones con el modelo cargado
predicciones_precios = modelo_cargado.predict(nuevas_superficies)

print("\n--- Predicciones con el modelo cargado ---")
for i, superficie in enumerate(nuevas_superficies):
    print(f"Precio estimado para una vivienda de {superficie[0]} m^2: {predicciones_precios[i]:.2f} ‚Ç¨")

# Verificamos que funciona igual que el original (si a√∫n lo tuvi√©ramos en memoria)
# prediccion_original_para_90 = modelo_precios_casas.predict(np.array([[90]]))
# print(f"Comparaci√≥n (si el original estuviera disponible): {prediccion_original_para_90[0]:.2f} ‚Ç¨")
```

Este proceso de guardar y cargar modelos es crucial para la puesta en producci√≥n de modelos de Machine Learning y para compartir modelos entre diferentes proyectos o colaboradores. `pickle` es una forma sencilla y efectiva de lograrlo en el ecosistema Python.
